# wordle-python
Wordle but in a CLI!

## Installation
```
git clone https://github.com/deciphr/wordle-python
```

## Usage
```
python3 main.py
```
or
```
python main.py
```

## How to Play
```
@ - Letter

---
|@| => This letter is not present at all
---

?-?
|@| => This letter is present, but isn't in its correct placement
?-?

+++
+@+ => This letter is present and is in its correct placement
+++
